Rise Against Lyrics Search

Enter a search term

Exact matches only

This app will search through all Rise Against songs (including covers) and display snippets of each song where a lyric matches the search term. Selecting "Exact matches only" will restrict the search to only the word(s) searched. Otherwise, it will display all lines that contain the search term. For example, by default it will include songs that contain the word "burning" when searching for "burn", and selecting "Exact matches only" will prevent this.
